## Description

> This project lets users move a blob throughout a continuous screen by using keyboard inputs (left and right arrow keys + holding space bar).  
> The player must navigate through obstacles that appear along the path.
> Design was inspired by Flappy Bird

---

## Interaction Instructions

Describe how to use or play your project.

The user start the game as soon as it loads. They use the left and right arrow keys to move horizontally and hold the space bar to go upward and fall slowly when its released. The goal is to avoid obstacles and stay within the screen as the world moves.
